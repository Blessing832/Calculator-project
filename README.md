Calculator Mini Project

This is a simple calculator built with **HTML**, **CSS**, and **JavaScript** as part of my October mini project for the Women Techsters Fellowship (Frontend track).

Features

- Basic operations: addition, subtraction, multiplication, division, percentage
- Clear (C) and backspace (⌫) buttons
- Prevents entering two operators in a row
- Responsive calculator card layout

Technologies Used

- HTML5
- CSS3 (Flexbox & CSS Grid)
- Vanilla JavaScript (DOM, event listeners, simple expression evaluation)

 How to Use

1. Open `index.html` in a browser, or visit the live GitHub Pages link.
2. Click the buttons to build an expression.
3. Press `=` to see the result.
4. Use `C` to clear and `⌫` to delete the last entry.
